# Queue

## Explanation
- FIFO: First In First Out

## Types of linked list
- Simple Queue
- Circular Queue
- Priority Queue
- Double Ended Queue

## Implementation
### Basic operations
- enqueue
- dequeue
- isEmpty
- isFull
- peek

### Helper Functions
- createQueue (in C)
- printQueue ( C & C++)

## Time Complexity
Considering implementation with array, it is O(1).
